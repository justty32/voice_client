# Mobile 版待改進事項

## 字體與縮放

- **字體太小**：需要提供字體大小調整功能（例如畫面上的 A-/A+ 按鈕，或設定面板裡的滑桿）
- 調整範圍建議：小 / 中 / 大 / 特大，或直接用 px 滑桿
- 設定值存入 `localStorage`，下次開啟自動套用

## 長按訊息編輯

- **長按 voice 或 user 訊息泡泡**，可將該筆文字載入下方輸入框進行編輯
- 編輯完後按送出（加入暫存），原訊息泡泡不變（僅複製內容，不是修改歷史）
- 實作方式：對每個訊息 DOM 加 `touchstart` + `touchend` 計時，超過 500ms 觸發；顯示簡短提示（例如輸入框 placeholder 改為「編輯中...」）讓使用者知道已載入

## 輸入框「送出」按鈕圖示

- 目前「▶」圖示語意不清楚——它其實是「加入暫存區」不是「立即送出」
- 改成堆疊感的圖示，例如：`＋` 、`⊕`、`📥`，或文字「入列」
- 行動按鈕列的「▶ 送出」（送出暫存區）才是真正的送出，兩者需要視覺上有明顯區別
