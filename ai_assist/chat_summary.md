# 2026年4月 AI 工具與框架精簡摘要

這份摘要整理自 2026 年 4 月的對話紀錄，重點在於解決原生 Gemini CLI 功能不足的問題，並針對「程式開發」與「日常行政」提供最強大的**開源**替代方案。

---

## 一、 核心概念：為什麼 2026 年的工具更聰明？
1. **MCP (Model Context Protocol):** 2026 年的標準協議，允許 AI 直接掛載工具（如搜尋、資料庫、Slack），不再需要手寫整合代碼。
2. **Repo Map (程式碼地圖):** 工具會自動壓縮並發送專案結構，讓 AI 理解跨檔案的引用關係，解決「健忘」問題。
3. **長效記憶 (Persistent Memory):** 透過 Mem0 等技術，AI 能記住使用者的習慣與偏好。

---

## 二、 程式開發類 (Coding & Analysis) - 推薦開源工具

### 1. 終端機 (CLI) 派
*   **Aider (首選標竿):** 
    *   **特色：** 自動 Git Commit、支援 Architect Mode (先設計再改 code)、Repo Map 深度理解專案。
    *   **優勢：** 最省 Token，且對 Gemini 1.5/3.1 Pro 的長上下文支援最穩。
*   **Claude Code 2.1:** 
    *   **特色：** Anthropic 官方 Agent，具備高度自主性，能自跑測試、修 Bug、掃描環境工具。

### 2. IDE 整合派
*   **Cline (原 Claude Dev):** 
    *   **特色：** VS Code 插件，支援「自主代理」模式。會擬定計畫並在執行前詢問，可直接操作終端機修復編譯錯誤。
*   **Continue:** 
    *   **特色：** 支援 VS Code 與 JetBrains，適合喜歡自定義斜槓指令與本地模型 (Ollama) 的開發者。

---

## 三、 行政庶務與知識管理 (Admin & Productivity) - 推薦開源工具

### 1. 快速自動化與文字處理
*   **Fabric 2.0:** 提供數百種「人類問題」模式（如：一鍵提取會議重點、將口語轉為正式公文）。
*   **Shell GPT (sgpt):** 輕量級工具，適合在終端機快速進行文字轉換或寫信。

### 2. 私人知識庫與文件管理
*   **AnythingLLM:** 桌面端工具，可建立多個工作區（如「財務報表」），將 PDF/Excel 丟入後進行精準問答。
*   **LobeHub:** 極精美的 Web UI 介面，支援豐富插件（行事曆、郵件、網頁爬蟲），適合管理非技術文件。
*   **Open WebUI:** 功能最完整的開源對話介面，支援多人協作與文件檢索。

---

## 四、 給開發者的框架與函式庫 (Library & Framework)

*   **CrewAI / LangGraph:** 用於建立「AI 專家團隊」與管理複雜、具備狀態的工作流。
*   **PydanticAI:** 2026 年最火紅框架，確保 AI 輸出的行政數據（如發票金額）型別安全且格式正確。
*   **Mem0:** 解決 AI 健忘的終極方案，為 AI 加入一層「智慧記憶體」，自動學習使用者偏好。
*   **Danswer:** 開源企業級搜尋，能自動同步並索引 Slack, Google Drive, Notion 等各處資料。

---

## 🚀 最終建議組合
*   **深度寫程式/分析原始碼：** 使用 **Aider**。
*   **日常行政與私人知識庫：** 使用 **LobeHub** 或 **AnythingLLM**。
*   **極客風自動化：** 使用 **Fabric**。
*   **解決 AI 健忘：** 在開發時引入 **Mem0**。
